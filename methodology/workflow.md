# Workflow

AI-LAB uses a minimal research workflow:

```text
Question
↓
Claim / Hypothesis
↓
Evidence / Experiment
↓
Result
↓
Spec / ADR / Pattern only if justified
```

## How To Use It

1. Start with a concrete question.
2. Create a claim or hypothesis only if the question can be verified.
3. Look for evidence in code, documentation, tests, releases, issues, PRs, discussions, community examples, or experiments.
4. If evidence is insufficient, design a minimal reproducible experiment.
5. Record the result and update the claim state.
6. Update specs, ADRs, or patterns only when the result is strong enough to justify it.

## Key Rule

> If there is not enough evidence, create a minimal reproducible experiment instead of closing the conclusion.
